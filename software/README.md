# Software - Memoria-Psiquiatrica-IA

Esta carpeta contiene las aplicaciones y herramientas de software del sistema.

## Contenido planificado

- `app-medico/` — aplicación para médicos (monitoreo, configuración, alertas).
- `app-paciente/` — aplicación para pacientes y cuidadores.
- `dashboard-hospital/` — panel de control para hospitales.
- `api/` — API para integración con sistemas de salud.

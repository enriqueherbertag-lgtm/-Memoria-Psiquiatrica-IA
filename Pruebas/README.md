# Pruebas - Memoria-Psiquiatrica-IA

Esta carpeta contiene las pruebas unitarias y de integración.

## Contenido planificado

- `test_sensors.py` — pruebas de los sensores (PPG, GSR, acelerómetro, temperatura).
- `test_detection.py` — pruebas del algoritmo de detección de crisis.
- `test_voice.py` — pruebas de reproducción de voz por conducción ósea.
- `test_connectivity.py` — pruebas de conectividad LTE-M y Bluetooth.

## Ejecutar pruebas

```bash
pytest tests/

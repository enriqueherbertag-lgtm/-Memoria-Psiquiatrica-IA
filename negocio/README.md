# Modelo de Negocio - Memoria Fija v2.1

## Visión General

Memoria Fija v2.1 se comercializará bajo un modelo **B2B (Business to Business)** dirigido a:
- Hospitales públicos y privados
- Clínicas psiquiátricas
- Sistemas de salud (ministerios, aseguradoras)
- Instituciones de investigación

---

## Propuesta de Valor

| Segmento | Propuesta |
|:---|:---|
| **Hospitales** | Reducción de rehospitalizaciones (60% → 20%), ahorro USD 12.400 por paciente/año |
| **Sistemas de salud** | Descompresión de urgencias psiquiátricas, monitoreo remoto de alta resolución |
| **Aseguradoras** | Menor costo de siniestralidad, mejor control de pacientes crónicos |
| **Pacientes** | Autonomía, tranquilidad, menos crisis |

## Estrategia de Go-to-Market

### Fase 1 (Años 1-2): Piloto en Chile
- Alianza con 3 hospitales públicos
- 30 pacientes piloto (estudio clínico)
- Validación de eficacia y modelo de negocio

### Fase 2 (Años 2-3): Expansión nacional
- Red de 10 hospitales
- 500 pacientes implantados
- Inicio de exportación a Latinoamérica

### Fase 3 (Años 4-5): Internacionalización
- Certificaciones FDA (EE.UU.) y CE (Europa)
- Distribuidores en México, Colombia, Brasil
- 2.000+ pacientes anuales
